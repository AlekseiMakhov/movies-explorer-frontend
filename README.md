# Фронтенд дипломного проекта "movies explorer"

### Функционал

* Редактирование профиля
* Редактирование аватара
* Сохранение фильма
* Удаление фильма
* Регистрация нового пользователя
* Авторизация пользователя

### Технологии

* Адаптивная верстка
* Flexbox
* Grid layout
* React
* Git
* Webpack
* npm
